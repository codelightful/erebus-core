Form

Utilities to handle forms and collect data from it

Methods

collect

Extracts all fields inside a HTMLElement container and collects its values into an object.

Each field to be collected should contain a model attribute with the name of the property that will hold the field value.

<div id="formContainer">
<input type="text" model="age"/>
</div>
<script type="text/javascript">
Erebus.form('#formContainer').collect();
</script> See running example

fill

Populates all the models fields inside a HTMLElement container with the values contained in an object instance.

Each field to be populated should contain a model attribute with the name of the property that will match the source object value.

<div id="formContainer">
<input type="text" model="age"/>
</div>
<script type="text/javascript">
Erebus.form('#formContainer').fill({ age: 35 });
</script> See running example
