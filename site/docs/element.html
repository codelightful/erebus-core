Element

Provides an instance to represent and wrap a DOM element to simplify manipulation and action on it

Creation

You can create a new element representing a specific block of HTML code var element = Erebus.element('<div&>This is a test</div>'); See running example The creation method refer an existing DOM element using a CSS selector var elementFromId = Erebus.element('#myElement');
var elementFromClass = Erebus.element('.myclass'); See running example Also, you can pass an existing HTMLElement var element = Erebus.element(document.body); See running example
Note: If you create an element refering to the body, a singleton instance will be created for it

Methods

Clear

Removes all the content inside the DOM elements represented by the instance.

Returns a reference to the element.

Erebus.element('#myElement').clear();

Content

Sets the content inside the element represented by the instance.

Returns a reference to the element.

The content can be a literal with plain text or HTML.

Example: This code snipplet sets a plain text content inside all the elements with the "cell" CSS class Erebus.element('#myElement').content('Some plain content');

The content can be a HTMLElement recently created or extracted from the DOM.

Example: This code snipplet sets the existing DOM element with the id "my-div" inside the element var sourceElement = document.getElementById('mydiv');
Erebus.element('#myElement').content(sourceElement);
