Formats

Utilities to format and transform data

Methods

format

Extracts all fields inside a HTMLElement container and collects its values into an object.

Each field to be collected should contain a model attribute with the name of the property that will hold the field value.

Erebus.formats('formatterCode').format(value); See running example

The available formatters are:

  • date: Formats a Date object or an string with a date representation in ISO format
  • datetime: Formats a Date object or an string with a date representation in ISO format
  • integer: Formats a numeric value without decimals
  • decimal: Formats a numeric value containing decimals
  • money: Formats a numeric value containing decimals adding group separator
